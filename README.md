Secret-Hitler-Telegram
===

A Telegram bot that allows users to play Secret Hitler.

## Special Commands

* Available at any time
  * /listplayers
  * /boardstats
  * /deckstats
  * /anarchystats
* ACCEPT_PLAYERS game state
  * /join
  * /leave
  * /startgame
* ELECTION game state
  * /blame lists all players who hasn't voted
  * /cancelvote
